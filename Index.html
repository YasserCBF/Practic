Mis Proyectos

Análisis de Datos con Python

Desarrollé un script en Python utilizando las librerías pandas y matplotlib para analizar un conjunto de datos de ventas. El programa limpia los datos, realiza cálculos estadísticos como promedios y tendencias, y genera visualizaciones interactivas como gráficos de líneas y barras. Implementé manejo de errores y optimicé el código para manejar grandes volúmenes de datos de manera eficiente.

Dashboard Educativo en React.js

Construí una aplicación web usando React.js, TypeScript y Tailwind CSS, basada en un prototipo de dashboard educativo con tema de bosque nevado. Incluye pantallas como Login Docente, Pantalla Principal y Registro de Alumnos, con navegación mediante React Router. Los componentes están organizados modularmente, y el diseño utiliza estilos responsivos para una experiencia visual atractiva.

Cuestionario de IoT y Electrónica

Diseñé un conjunto de preguntas tipo quiz sobre Internet of Things (IoT) y electrónica básica, demostrando conocimientos en programación para plataformas como Arduino. Las preguntas cubren conceptos como señales analógicas y digitales, uso de multímetros, y desarrollo en entornos IoT, reflejando mi comprensión de tecnologías emergentes.
